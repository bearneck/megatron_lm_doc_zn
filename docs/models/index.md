# 支持的模型

Megatron Core 支持广泛的单模态语言模型和多模态模型，并针对大规模训练提供了优化的实现。

## 模型转换

如需将 HuggingFace 模型转换为 Megatron 格式，请使用官方的独立转换工具 [Megatron Bridge](https://github.com/NVIDIA-NeMo/Megatron-Bridge)。Megatron Bridge 支持广泛的模型列表，包括 LLaMA、Mistral、Mixtral、Qwen、DeepSeek、Gemma、Phi、Nemotron 等众多模型。

有关支持的模型的完整和最新列表，请参阅 [Megatron Bridge 支持的模型列表](https://github.com/NVIDIA-NeMo/Megatron-Bridge?tab=readme-ov-file#supported-models)。
